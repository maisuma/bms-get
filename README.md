# bms-get

BMS用自動譜面ダウンロードツール  
md5、難易度表、イベントページからのダウンロードに対応しています。

## 機能
- BMS SEARCH API / LR2IR を利用したmd5ハッシュからの譜面取得
- 難易度表に含まれる全譜面の一括取得
- イベントサイトに登録された全譜面の一括取得
- 保存したアーカイブの自動解凍機能

## 使用方法

特定のmd5からダウンロード
```
bms-get md5 <MD5>
```

難易度表からダウンロード
```
bms-get table <URL>
```

イベントページからダウンロード
```
bms-get event <URL>
```
